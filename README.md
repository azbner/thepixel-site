# the PIXEL site

Site statique pour GitHub Pages.

## Fichiers principaux

- `index.html`
- `styles.css`
- `script.js`

## Publication rapide sur GitHub Pages

1. Cree un depot GitHub.
2. Envoie ces fichiers dans le depot.
3. Active GitHub Pages dans `Settings > Pages`.
4. Choisis la branche `main` et le dossier `/root`.
5. Attends que GitHub donne l'URL publique.
